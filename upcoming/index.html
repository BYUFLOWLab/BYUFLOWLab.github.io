Prospective Students

Below I list some resources to help prospective students get up to speed. Incoming students are certainly not expected to be familiar with all of the tools or materials we use, but developing familiarity with some of these would be helpful. You are welcome to attend our group meetings to learn more about what we are working on. Come talk to me if you are interested in contributing to an existing project or have an new idea that you are interested in exploring.

  • There is a useful online course on High Performance Scientific Computing that you can access for free: (Coursera course, course notes). It covers many relevant topics like version control, Python, Fortran, MPI. Most of us use Python (v2) as our primary programming language. We also use Matlab, Julia, Fortran, and C/C++. We use Git for version control and LaTeX for technical writing.

  • For those interested in computational fluid dynamics, we are primarily using StarCCM+. There are good tutorials available on the CAEDM computers.

  • A few good introductory books on topics relevant to our research:
    • Aerodynamics: Fundamentals of Aerodynamics, John Anderson
    • Optimization: Multidisciplinary Design Optimization, Joaquim Martins
    • Wind Energy: Aerodynamics of Wind Turbines, Martin Hansen
    • Flight: Fundamentals of Flight, Richard Shevell
