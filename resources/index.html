Prospective Students

You are welcome to attend our group meetings to learn more about what we are working on (currently Thursday at 10AM in 431A CTB). If you are interested in contributing to an existing project or have an idea that you are interested in exploring, send me your resume and a one page statement of interest. Most of our projects involve scientific computing so it would be helpful to be comfortable with a programming language. Any language is fine but we primarily use the following (in order of frequency of use): Python (v2), Matlab, Julia, Fortran, C, C++. We also use Git quite a bit for version control. A useful resource to learn more about High Performance Scientific Computing is available here: Coursera course, course notes. If you are interested in CFD, you should work through some of the StarCCM+ tutorials that are available in the CAEDM lab.

Beginning Researchers

Your goal as a researcher is to make an impact for good in the world and improve the way we design wind and flight systems. The primary way to make an impact is through tackling challenging and important questions, performing high-quality, insightful work, collaborating with industry and other researchers, and sharing findings broadly at conferences and in peer-reviewed journals.
