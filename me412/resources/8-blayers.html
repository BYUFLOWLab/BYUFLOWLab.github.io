Boundary Layer Displacement Thickness (\(\delta^*\)): Distance we would need to displace the wall so that an exterior inviscid flow (i.e., at \(U_e\)) would have the same mass flux as that actual flow.

Let \(h\) be some distance far outside the boundary layer, and consider the mass flow rate per unit depth.

Mass flow rate through fictitious inviscid flow: \[\rho U_e (h - \delta^*)\]

Mass flow rate through actual flow: \[\int_0^h \rho u dy\]

Equate and solve: \[\delta^* = \int_0^h \left(1 - \frac{\rho u}{\rho_e U_e}\right) dy\]

The location \(h\) was arbitrary so we let \(h \rightarrow \infty\): \[ \boxed{ \delta^* = \int_0^\infty \left(1 - \frac{\rho u}{\rho_e U_e}\right) dy } \]

Alternative derivation

\(\delta^*\) can also be interpreted as the distance streamlines outside the boundary layer are displaced due to the presence of the boundary layer.

For a flat plate

upstream: \[\int_0^h \rho_e U_e dy\]

downstream: \[\int_0^h \rho u dy + \rho_e U_e \delta^*\]

Yields same solution.

keep in mind these are all definitions

A practical use of \(\delta^*\)

\(\delta^*\) defines an effective body shape.

  1. Analyze inviscid flow and find \(U_e\) along surface.
  2. Solve boundary layer equations and compute \(\delta^*\).
  3. Reanalyze inviscid flow, but use \(\delta^*\) in flow tangency boundary condition.
  4. Repeat until convergence.

Boundary Layer Momentum Thickness (\(\Theta\)): Distance we would need to displace the wall so that an exterior inviscid flow (i.e., at \(U_e\)) would have the same momentum flux as that actual flow.

momentum deficit in fictitious flow: \[ \rho_e U_e^2 \Theta \]

momentum deficit in actual flow: \[ \int_0^h \rho u (U_e- u) dy \]

\[ \Theta = \int_0^\infty \frac{\rho u}{\rho_e U_e} \left( 1 - \frac{u}{U_e} \right)dy \]

Why is this useful?

Momentum thickness is useful in estimating skin friction drag coefficient (more on this later).
